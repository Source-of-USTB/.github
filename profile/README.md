<h2>索思科技协会</h2>

</div>

**全心求索，同道而思**

索思科技协会成立于1991年，历经三十余载沉淀， 已成长为北科历史最悠久、影响力最为深远的学术科技类社团之一。

下设有办公室、学术部、宣传部和技术部。
